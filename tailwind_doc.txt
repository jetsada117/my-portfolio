=====================================================================
                 TAILWIND CSS CHEAT SHEET (DETAILED & UPDATED)
=====================================================================
* กฎพื้นฐาน: ระบบขนาดของ Tailwind มักจะคูณด้วย 4px (เช่น 4 = 16px, 8 = 32px)
* สีพื้นฐาน: มีระดับความเข้มตั้งแต่ 50 (อ่อนสุด) ถึง 900 (เข้มสุด)

---------------------------------------------------------------------
1. SPACING (การจัดการระยะห่าง - Margin & Padding)
---------------------------------------------------------------------
[Margin - ระยะห่างด้านนอก]
m-0     : margin 0px
m-4     : margin 4 ด้าน (16px)
mx-auto : จัดให้อยู่กึ่งกลาง (margin left/right auto)
mx-4    : margin ซ้ายและขวา 16px
my-4    : margin บนและล่าง 16px
mt-4    : margin ด้านบน (top)
mb-4    : margin ด้านล่าง (bottom)
ml-4    : margin ด้านซ้าย (left)
mr-4    : margin ด้านขวา (right)

[Padding - ระยะห่างด้านใน]
p-0     : padding 0px
p-4     : padding 4 ด้าน (16px)
px-4    : padding ซ้ายและขวา 16px
py-4    : padding บนและล่าง 16px
pt-4    : padding ด้านบน (top)
pb-4    : padding ด้านล่าง (bottom)
pl-4    : padding ด้านซ้าย (left)
pr-4    : padding ด้านขวา (right)

---------------------------------------------------------------------
2. SIZING (การกำหนดขนาด - Width & Height)
---------------------------------------------------------------------
[Width - ความกว้าง]
w-full    : กว้าง 100% ของคอนเทนเนอร์แม่
w-screen  : กว้าง 100% ของหน้าจอ (Viewport Width)
w-1/2     : กว้าง 50%
w-1/3     : กว้าง 33.333%
w-auto    : กว้างตามเนื้อหาอัตโนมัติ
w-32      : กว้างตายตัว 128px (32 * 4)
max-w-md  : กว้างสูงสุดระดับ Medium (28rem / 448px)

[Height - ความสูง]
h-full    : สูง 100% ของคอนเทนเนอร์แม่
h-screen  : สูง 100vh ของหน้าจอ
h-auto    : สูงตามเนื้อหาอัตโนมัติ
h-32      : สูงตายตัว 128px
min-h-screen : สูงขั้นต่ำเท่ากับหน้าจอเสมอ

---------------------------------------------------------------------
3. LAYOUT & POSITION (การจัดวางและตำแหน่ง)
---------------------------------------------------------------------
[Display]
block        : แสดงผลแบบ Block
inline-block : แสดงผลแบบ Inline Block
flex         : เปิดใช้ Flexbox
grid         : เปิดใช้ Grid
hidden       : ซ่อนองค์ประกอบ (display: none)

[Position]
relative     : จัดตำแหน่งอ้างอิงจากตำแหน่งเดิม
absolute     : จัดตำแหน่งอิสระ (มักใช้กับ top-0, left-0, bottom-0, right-0)
fixed        : ล็อกติดหน้าจอเวลาเลื่อนสกอร์ (เช่น Navbar)
sticky       : ล็อกติดหน้าจอเมื่อเลื่อนมาถึงจุดที่กำหนด
z-0 ถึง z-50   : ลำดับการซ้อนทับ (z-index) ยิ่งมากยิ่งอยู่บน

---------------------------------------------------------------------
4. FLEXBOX & GRID (การจัดเรียงไอเทม)
---------------------------------------------------------------------
[Flexbox]
flex-row        : เรียงแนวนอน (ซ้ายไปขวา) - ค่าเริ่มต้น
flex-col        : เรียงแนวตั้ง (บนลงล่าง)
justify-start   : ชิดซ้าย / ชิดบน
justify-center  : จัดกึ่งกลางแกนหลัก
justify-between : ดันไอเทมชิดขอบสองข้าง ระยะห่างตรงกลางเท่ากัน
justify-end     : ชิดขวา / ชิดล่าง
items-start     : จัดชิดบน (แกนรอง)
items-center    : จัดกึ่งกลาง (แกนรอง)
items-stretch   : ขยายให้เต็มพื้นที่ (แกนรอง)
flex-wrap       : ตัดขึ้นบรรทัดใหม่เมื่อเนื้อหาล้น

[Grid]
grid-cols-1 ถึง 12 : แบ่งจำนวนคอลัมน์ (เช่น grid-cols-3 คือแบ่ง 3 คอลัมน์)
gap-4             : ระยะห่างระหว่างช่องทั้งหมด 16px
gap-x-4           : ระยะห่างระหว่างคอลัมน์ (แนวนอน)
gap-y-4           : ระยะห่างระหว่างแถว (แนวตั้ง)

---------------------------------------------------------------------
5. TYPOGRAPHY (ตัวอักษรและข้อความ)
---------------------------------------------------------------------
[Font Size]
text-xs   : ขนาดเล็กมาก (12px)
text-sm   : ขนาดเล็ก (14px)
text-base : ขนาดมาตรฐาน (16px)
text-lg   : ขนาดใหญ่ (18px)
text-xl   : 20px
text-2xl  : 24px (ไปจนถึง 9xl)

[Font Weight]
font-light    : บาง (300)
font-normal   : ปกติ (400)
font-medium   : หนาปานกลาง (500)
font-semibold : หนา (600)
font-bold     : หนามาก (700)
font-extrabold: หนาพิเศษ (800)

[Text Alignment & Style]
text-left     : จัดชิดซ้าย
text-center   : จัดกึ่งกลาง
text-right    : จัดชิดขวา
text-justify  : จัดกระจายเต็มบรรทัด
uppercase     : พิมพ์ใหญ่ทั้งหมด
capitalize    : พิมพ์ใหญ่ตัวแรกของคำ
underline     : ขีดเส้นใต้
line-through  : ขีดฆ่า
truncate      : ตัดคำที่ล้นด้วย ... (จุดไข่ปลา)

[Text Colors]
text-black, text-white, text-transparent
text-gray-500, text-blue-600, text-red-500 (ใช้สีและระดับความเข้ม)

---------------------------------------------------------------------
6. BACKGROUND, BORDER, SHADOW (พื้นหลัง เส้นขอบ เงา)
---------------------------------------------------------------------
[Background]
bg-white, bg-black, bg-transparent
bg-blue-500   : พื้นหลังสีน้ำเงิน
bg-cover      : ปรับภาพพื้นหลังให้เต็มพื้นที่
bg-center     : จัดกึ่งกลางภาพพื้นหลัง

[Border Radius - ความโค้งมน]
rounded-sm    : โค้งเล็กน้อย
rounded-md    : โค้งปานกลาง
rounded-lg    : โค้งมาก (8px)
rounded-xl    : โค้งพิเศษ (12px)
rounded-2xl   : โค้งพิเศษมาก (16px)
rounded-3xl   : โค้งวงกลมใหญ่ (24px)
rounded-full  : โค้งเป็นวงกลม / แคปซูล
rounded-t-lg  : โค้งเฉพาะด้านบน (top)

[Border Width & Color]
border        : เปิดเส้นขอบหนา 1px 4 ด้าน
border-2      : เส้นขอบหนา 2px
border-t      : เส้นขอบเฉพาะด้านบน
border-blue-500 : สีเส้นขอบ

[Shadow - เงา]
shadow-sm     : เงาเล็ก (เหมาะกับการ์ดเล็กๆ)
shadow-md     : เงากลาง (ใช้บ่อยสุด)
shadow-lg     : เงาใหญ่ (เหมาะกับ Modal/Popup)
shadow-xl     : เงาใหญ่มาก
shadow-none   : ไม่มีเงา

---------------------------------------------------------------------
7. STATES & TRANSITIONS (สถานะและแอนิเมชัน)
---------------------------------------------------------------------
hover:bg-blue-600    : เปลี่ยนสีพื้นหลังเมื่อเมาส์ชี้
focus:ring-2         : แสดงวงแหวนโฟกัส (เมื่อคลิก/พิมพ์ช่อง Input)
focus:outline-none   : ปิดเส้น Outline สีฟ้าเริ่มต้นของเบราว์เซอร์
active:scale-95      : ย่อขนาดลง 5% เมื่อกดคลิก

[Transitions]
transition-all       : เปิดใช้ Transition สำหรับทุก Property
duration-300         : ใช้เวลาเปลี่ยนผ่าน 300ms
ease-in-out          : รูปแบบความเร็วการเปลี่ยนผ่าน

---------------------------------------------------------------------
8. RESPONSIVE DESIGN (การรองรับขนาดหน้าจอ)
---------------------------------------------------------------------
* Tailwind ใช้แนวคิด Mobile First (กำหนดค่าสำหรับมือถือก่อน แล้วใส่ Prefix สำหรับจอที่ใหญ่ขึ้น)

(ไม่มี Prefix)  : หน้าจอมือถือ (เริ่มต้น)
sm:            : จอขนาด >= 640px (แท็บเล็ตขนาดเล็ก)
md:            : จอขนาด >= 768px (แท็บเล็ต/ไอแพด)
lg:            : จอขนาด >= 1024px (แล็ปท็อป)
xl:            : จอขนาด >= 1280px (เดสก์ท็อป)
2xl:           : จอขนาด >= 1536px (จอใหญ่พิเศษ)

ตัวอย่าง: w-full md:w-1/2 lg:w-1/4
ความหมาย: มือถือกว้าง 100%, จอขนาดกลางกว้าง 50%, จอคอมกว้าง 25%

9. ARBITRARY VALUES (การกำหนดค่าอิสระแบบระบุเองด้วยวงเล็บ [ ])
---------------------------------------------------------------------
* ใช้เมื่อต้องการระบุขนาด สี หรือตำแหน่งที่ไม่พบบนสเกลมาตรฐานของ Tailwind
[ตัวอย่างการใช้งาน]
w-[60px]      : กำหนดความกว้าง 60px ตรงๆ (เลี่ยงข้อผิดพลาด w-15)
h-[70px]      : กำหนดความสูง 70px
bg-[#2563eb]  : กำหนดสีพื้นหลัง Hex Code สีน้ำเงิน
text-[32px]   : กำหนดขนาดตัวอักษร 32px
z-[1000]      : กำหนดค่า z-index เท่ากับ 1000 (ใช้ล็อกเมนู Navbar ให้อยู่บนสุด)
grid-cols-[1.2fr_0.8fr] : กำหนดขนาดคอลัมน์ Grid ด้วยอัตราส่วนเฉพาะตัว

---------------------------------------------------------------------
10. GRADIENTS & COLOR CLIPPING (การไล่เฉดสี และตัวอักษรโปร่งแสง)
---------------------------------------------------------------------
* การไล่เฉดสีสร้างแสงและเงาให้เว็บไซต์หรูหราขึ้น
bg-gradient-to-r  : ไล่เฉดสีจากซ้ายไปขวา
bg-gradient-to-br : ไล่เฉดสีจากบนซ้ายไปล่างขวา
from-blue-600     : กำหนดสีเริ่มต้น (จากน้ำเงิน)
via-indigo-600    : กำหนดสีตรงกลาง (ผ่านคราม)
to-violet-600     : กำหนดสีปลายทาง (ไปยังม่วง)

[Gradient Text - ตัวอักษรสีไล่ระดับ]
* วิธีทำตัวอักษรไล่ระดับสีสวยๆ:
bg-clip-text text-transparent bg-gradient-to-r from-blue-600 to-indigo-600
(ความหมาย: ทำตัวหนังสือโปร่งแสง แล้วตัดเฉดสีพื้นหลังให้เข้าตามรูปทรงตัวหนังสือ)

---------------------------------------------------------------------
11. FILTERS & GLASSMORPHISM (การกรองความเบลอ และเอฟเฟกต์แผ่นกระจก)
---------------------------------------------------------------------
[Blur - การเบลอพื้นหลังสร้างมิติความลึก (Glow Blob)]
blur-sm, blur-md, blur-lg, blur-xl, blur-[130px]
* ตัวอย่าง: ทำดวงไฟพื้นหลังเรืองแสงมีมิติ:
"absolute rounded-full bg-blue-300/15 blur-[130px] pointer-events-none"

[Backdrop Blur - เอฟเฟกต์การ์ดกระจกแก้ว (Glassmorphic Card)]
backdrop-blur-sm, backdrop-blur-md, backdrop-blur-lg
* ตัวอย่าง: การ์ดกระจกหรูหราสะท้อนเงา:
"bg-white/70 backdrop-blur-sm border border-slate-200/60 shadow-md"
* หมายเหตุ: ต้องใช้คู่กับสีพื้นหลังที่โปร่งแสงด้วยเสมอ เช่น bg-white/70, bg-slate-50/80

---------------------------------------------------------------------
12. TRANSFORMS & SCALING (การเลื่อน ย่อขยาย และลูกเล่นเชิงโต้ตอบ)
---------------------------------------------------------------------
* ใช้สร้าง Micro-interaction เวลาเมาส์ชี้เพื่อให้ผู้ใช้รู้สึกว่าเว็บตอบสนองได้ดี
scale-95      : ย่อขนาดวัตถุเหลือ 95%
scale-100     : ขนาดปกติ 100%
scale-105     : ขยายวัตถุขึ้นเป็น 105%
scale-[1.01]  : ขยายวัตถุเล็กน้อย 1% (แบบหรูหราพรีเมียม)
-translate-y-2: เลื่อนวัตถุขึ้นด้านบน 8px
translate-x-2 : เลื่อนวัตถุไปด้านขวา 8px
rotate-45     : หมุนวัตถุ 45 องศา

* ตัวอย่าง: ชี้เมาส์แล้วการ์ดลอยขึ้นและมีเงาเข้มขึ้น:
"transition-all duration-300 hover:-translate-y-2 hover:shadow-lg"

---------------------------------------------------------------------
13. DEFAULT ANIMATIONS (แอนิเมชันสำเร็จรูปของ Tailwind)
---------------------------------------------------------------------
animate-spin   : หมุน 360 องศาไปเรื่อยๆ (เหมาะสำหรับไอคอนโหลดไอคอนหมุน)
animate-ping   : แสดงวงสัญญาณขยายตัวและจางลง (เหมาะสำหรับจุดเตือนสถานะความเคลื่อนไหว)
animate-pulse  : กะพริบจางหายสลับขึ้นมาเรื่อยๆ (เหมาะทำ Skeleton screen หรือ Badge ประดับปุ่ม)
animate-bounce : วัตถุกระโดดขึ้นลงเป็นจังหวะ (เหมาะสำหรับลูกศรชี้ลงล่าง หรือไอคอน 🎯)

---------------------------------------------------------------------
14. ASPECT RATIO (การล็อกสัดส่วนขนาดภาพ)
---------------------------------------------------------------------
aspect-auto    : ขนาดอัตราส่วนตามภาพจริง
aspect-square  : ล็อกอัตราส่วนจัตุรัส 1:1 เสมอ (เหมาะกับรูปโปรไฟล์ / การ์ดโปรเจกต์)
aspect-video   : ล็อกอัตราส่วนแนวนอน 16:9 เสมอ (เหมาะกับหน้าจอวิดีโอ / รูปปก)
